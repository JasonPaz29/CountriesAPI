Applicable Functions for API

This section lists all applicable functions for the Countries API

  • get_populated_countries - Retrieve countries sorted by population
    • Description: Get a list of countries ordered by population size
    • Parameters: sort_order (ascending/descending), limit (number of results)
    • Example use: Find the top 10 most populated countries
  • get_countries_capital - Get capital cities for all countries

    Returns a complete list of all countries with their capital cities. Useful for creating dropdown menus or reference tables.

    Returns: Array of country-capital pairs

  • compare_countries - Compare statistics between two countries
    • Description: Side-by-side comparison of key metrics
    • Parameters: country1 (string), country2 (string)
    • Returns: HDI, population, economy type, language, capital for both countries
    • Example use: Compare Canada vs Japan economic indicators
  • get_single_country - Get detailed data for one country

    • Retrieve all available data for a specific country by name.

    • Parameters: country_name (string)

    • Returns: Complete country profile including economy type, language, HDI, population, and capital

  • get_countries_with_filter - Filter countries by HDI, economy type, or language
    • Description: Search for countries matching specific criteria
    • Parameters: filter_type (HDI/economy/language), filter_value (the value to match)
    • Example use: Find all countries with "Developed" economy or HDI above 0.9
